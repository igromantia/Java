Задание 7
Необходимо создать два интерфейса Strings(струнные) и Keyboards(Kлавишные).
Инструменты Гитара(Guitar), скрипка(Violin), Виолончель(Cello), Басс(Bass) должны
имплементировать интерфейс струнные.
Инструменты Пианино(Piano), Орган(Organ), Клавесин(Harpsichord) должны
имплементировать интерфейс клавишные.
У интерфейса Strings должен быть один void метод playStrings().
Этот метод выводит на консоль строку о том, что данный инструмент играет свою
партию.
Например результатом работы этого метода у класса Guitar должна быть строка
«Играет гитара.»
У интерфейса Keyboards должен быть один void метод playKeys().
Этот метод выводит на консоль строку о том, что данный инструмент играет свою
партию.
Например результатом работы этого метода у класса piano должна быть строка
«Играет Пианино.»
Необходимо также создать два класса-оркестра: струнный(StringOrchestra) и
клавишный(KeysOrchestra).
Каждый класс должен содержать в себе поля типов инструментов, которые должны
играть в оркестре. В струнном струнные инструменты, а в клавишном клавишные.
Каждый класс оркестра должен содержать конструктор без параметров в котором
инициализируются все его поля. Каждый класс оркестра должен содержать метод
playYourParts() - играйте свои партии. В этом методе у каждого элемента класса
инструмента должен вызываться метод playStrings или playKeys в зависимости от того
интерфейса, который класс реализует.
